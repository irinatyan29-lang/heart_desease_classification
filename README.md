# heart_desease_classification
Проект прогнозирования наличия сердечно-сосудистых заболеваний на основе клинических данных пациентов. Задача сформулирована как задача бинарной классификации, в которой модель определяет, находится ли пациент в группе риска.  Проект демонстрирует полный цикл машинного обучения: от анализа данных до оценки и интерпретации модели.
