# heart_desease_classification
# Прогнозирование сердечно-сосудистых заболеваний

## Описание проекта

Данный проект посвящён прогнозированию наличия сердечно-сосудистых заболеваний на основе клинических данных пациентов. Задача сформулирована как задача бинарной классификации, в рамках которой модель определяет, находится ли пациент в группе риска.

Проект демонстрирует полный цикл разработки модели машинного обучения: от первичного анализа данных до оценки качества модели и интерпретации результатов.


## Описание признаков

В датасете используются следующие признаки:

**age**: возраст пациента
**sex**: пол (1 = мужчина, 0 = женщина)

**cp**: тип боли в груди
0 типичная стенокардия
1 атипичная стенокардия
2 неангинозная боль
3 бессимптомное течение

**trestbps**: артериальное давление в состоянии покоя (мм рт. ст.)
**chol**: уровень холестерина в крови (мг/дл)

**fbs**: уровень сахара в крови > 120 мг/дл
1 да
0 нет

**restecg**: результаты ЭКГ в состоянии покоя
0 норма
1 наличие изменений ST-T
2 признаки гипертрофии левого желудочка

**thalach**: максимальная достигнутая частота сердечных сокращений

**exang**: стенокардия при физической нагрузке
1 да
0 нет

**oldpeak**: депрессия ST, вызванная нагрузкой относительно состояния покоя

**slope**: наклон ST-сегмента
0 восходящий
1 горизонтальный
2 нисходящий

**ca**: количество крупных сосудов (0–3), окрашенных при флюороскопии

**thal**: результат теста с таллием
1 нормальный
2 фиксированный дефект
3 обратимый дефект

**target**: целевая переменная
1 наличие сердечного заболевания
0 отсутствие

* Разведочный анализ данных (EDA)
